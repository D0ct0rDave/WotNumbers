493 DbVersion.cs:
-----------------
Use this file to modify programatically Database data, both modify the database structure (like adding a new field on a table) or updating a table with new values, or changing the values of a table row. ExpectedNumber holds the value olf the version of the database.

1566 createTableSQLite.txt :
----------------------------
Database creation scripts. This file is uses when the application is installed for the first time and no DB exists. DONT ADD NEW FIELDS if new fields are created. They will be added programatically in DbVersion.cs

1568 insert_dbver488.txt
----------------------------
Initial data inserted into the tables created according to createTableSQLite. Do not add data here if it is already added in DbVersion.cs

2428 VersionSettings.json
-------------------------
This file holds the latest version of the application. This file is stored at a remote site too (for instance GitHub), On application startup the app will check the file in the remote site to see if a new version was deployed.
